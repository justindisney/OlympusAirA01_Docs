TAKEMODE - Shooting Modes

Primary mode selector that affects which other properties are available. Always set TAKEMODE first.

Value Mode Description Available Properties
<TAKEMODE/iAuto> Intelligent Auto Camera auto-selects all settings Limited control
<TAKEMODE/P> Program Camera sets aperture & shutter Most properties
<TAKEMODE/A> Aperture Priority User sets aperture APERTURE control
<TAKEMODE/S> Shutter Priority User sets shutter speed SHUTTER control
<TAKEMODE/M> Manual Full manual control APERTURE + SHUTTER
<TAKEMODE/ART> Art Filter Creative filters Art filter properties
<TAKEMODE/movie> Movie Video recording Movie properties, FOCUS_MOVIE

COLORTONE - Picture Modes

22 valid values confirmed from PlayOPC source code

Basic Picture Modes (8)

  • <COLORTONE/NATURAL> - Balanced natural colors (default)
  • <COLORTONE/VIVID> - Enhanced saturation and contrast
  • <COLORTONE/FLAT> - Low contrast for post-processing (Muted)
  • <COLORTONE/Monotone> - Black & white
  • <COLORTONE/I_FINISH> - Olympus intelligent mode (i-Enhance)
  • <COLORTONE/Portrait> - Portrait optimized
  • <COLORTONE/ePortrait> - Enhanced portrait
  • <COLORTONE/COLOR_CREATOR> - Advanced color control

Art Filter Modes (14)

  • <COLORTONE/POPART> - Pop Art (bold vivid colors)
  • <COLORTONE/FANTASIC_FOCUS> - Soft Focus (dreamy ethereal)
  • <COLORTONE/DAYDREAM> - Pale & Light Color / Watercolor (soft watercolor painting)
  • <COLORTONE/LIGHT_TONE> - Light Tone (bright airy)
  • <COLORTONE/ROUGH_MONOCHROME> - Grainy Film (high contrast B&W)
  • <COLORTONE/TOY_PHOTO> - Pin Hole (vignetting + saturated)
  • <COLORTONE/MINIATURE> - Diorama (miniature model effect)
  • <COLORTONE/CROSS_PROCESS> - Cross Process (unusual colors)
  • <COLORTONE/GENTLE_SEPIA> - Gentle Sepia (soft sepia tone)
  • <COLORTONE/DRAMATIC_TONE> - Dramatic Tone (high contrast detail)
  • <COLORTONE/LIGNE_CLAIR> - Key Line (simplified colors + outlines)
  • <COLORTONE/PASTEL> - Watercolor (soft pastels)
  • <COLORTONE/VINTAGE> - Vintage (aged photograph)
  • <COLORTONE/PARTCOLOR> - Partial Color (selective color)
Note: SOFT is NOT a valid COLORTONE value. Properties like CONTRAST_SOFT, SHARP_SOFT, SATURATION_LEVEL_SOFT, and TONE_SOFT exist in the API but do not correspond to any actual picture mode and appear to be legacy/unused.

Picture Mode Adjustments (Contrast, Sharpness, Saturation)

Each picture mode has its own set of adjustable parameters for fine-tuning the look. All adjustments use a -2 to +2 scale.

Available Properties by Picture Mode

Picture Mode Contrast Sharpness Saturation
NATURAL CONTRAST_NATURAL SHARP_NATURAL SATURATION_LEVEL_NATURAL
VIVID CONTRAST_VIVID SHARP_VIVID SATURATION_LEVEL_VIVID
FLAT CONTRAST_FLAT SHARP_FLAT SATURATION_LEVEL_FLAT
I_FINISH CONTRAST_I_FINISH SHARP_I_FINISH SATURATION_LEVEL_I_FINISH
Monotone CONTRAST_MONOCHROME SHARP_MONOCHROME N/A (B&W mode)

Value Range

All properties accept values from -2 (minimum) to +2 (maximum):

  • -2 - Minimum (softest/lowest)
  • -1 - Reduced
  • 0 - Standard/neutral
  • +1 - Enhanced
  • +2 - Maximum (hardest/highest)

Usage Examples

# Soft, low-contrast natural look
<COLORTONE/NATURAL>
<CONTRAST_NATURAL/-2>
<SHARP_NATURAL/-1>
<SATURATION_LEVEL_NATURAL/0>

# Punchy, vivid look with enhanced sharpness
<COLORTONE/VIVID>
<CONTRAST_VIVID/+2>
<SHARP_VIVID/+2>
<SATURATION_LEVEL_VIVID/+1>

# Flat profile optimized for post-processing
<COLORTONE/FLAT>
<CONTRAST_FLAT/-2>
<SHARP_FLAT/-1>
<SATURATION_LEVEL_FLAT/0>

# High-contrast B&W
<COLORTONE/Monotone>
<CONTRAST_MONOCHROME/+2>
<SHARP_MONOCHROME/+1>
Note: CONTRAST_SOFT, SHARP_SOFT, and SATURATION_LEVEL_SOFT exist in the API but are not usable since SOFT is not a valid COLORTONE mode.

TONE - Gradation (Tone Curve Presets)

Gradation controls the overall tonal distribution of the image using preset tone curves. Each picture mode has its own TONE property.

Important: TONE properties (gradation) are different from TONE_CONTROL properties (shadow/midtone/highlight). Gradation sets the overall tone curve preset, while tone control provides fine-tuning adjustments.

Available TONE Properties

Property Required COLORTONE Values
TONE_NATURAL NATURAL AUTO, NORMAL, HIGHKEY, LOWKEY
TONE_VIVID VIVID AUTO, NORMAL, HIGHKEY, LOWKEY
TONE_FLAT FLAT AUTO, NORMAL, HIGHKEY, LOWKEY
TONE_I_FINISH I_FINISH AUTO, NORMAL, HIGHKEY, LOWKEY
TONE_MONOCHROME Monotone AUTO, NORMAL, HIGHKEY, LOWKEY

Gradation Values

  • AUTO - Camera automatically selects tone curve
  • NORMAL - Standard tone curve distribution
  • HIGHKEY - High Key (brightens overall, expands highlight range)
  • LOWKEY - Low Key (darkens overall, expands shadow range)

Usage Examples

# High-key natural look (bright and airy)
<COLORTONE/NATURAL>
<TONE_NATURAL/HIGHKEY>

# Low-key vivid look (moody and dramatic)
<COLORTONE/VIVID>
<TONE_VIVID/LOWKEY>

# Flat profile for post-processing
<COLORTONE/FLAT>
<TONE_FLAT/NORMAL>

# High-key black & white
<COLORTONE/Monotone>
<TONE_MONOCHROME/HIGHKEY>
Use Case: Gradation is ideal for controlling the overall mood and tonal range. Use HIGHKEY for bright, airy portraits and LOWKEY for dramatic, moody scenes. Combine with TONE_CONTROL properties for precise tonal adjustments.

Art Filters

Art filters are available in TAKEMODE=ART. Set RECENTLY_ART_FILTER to select which filter to use.

Each filter supports:

  1. Type variations - ART_EFFECT_TYPE_[FILTER] (TYPE1, TYPE2, TYPE3, etc.)
  2. Hybrid effects - ART_EFFECT_HYBRID_[FILTER] (combine multiple effects)

Art Bracket Mode

  • Set RECENTLY_ART_FILTER to ART_BKT
  • Enable desired filters using BRACKET_PICT_* properties
  • One shutter press captures multiple images with different filters
<RECENTLY_ART_FILTER/ART_BKT>
<BRACKET_PICT_DRAMATIC_TONE/ON>
<BRACKET_PICT_VINTAGE/ON>
<BRACKET_PICT_ROUGH_MONOCHROME/ON>
<BRACKET_PICT_TOY_PHOTO/ON>
<BRACKET_PICT_PARTCOLOR/ON>

Art Filter Hybrid Effects

Add secondary effects to art filters. Available hybrids vary by base filter.

Common Hybrid Effects

  • OFF - No hybrid effect
  • FANTASIC_FOCUS - Add soft focus overlay
  • TOY_PHOTO - Add vignetting/darkened edges
  • WHITE_EDGE - Add white border edges
  • FRAME_JAGGY - Add jagged frame effect
  • STARLIGHT - Add star/sparkle effects to lights
  • MINIATURE_VERTICAL - Vertical selective blur (diorama effect)
  • MINIATURE_HORIZON - Horizontal selective blur (diorama effect)
  • SHADING_HORIZON - Horizontal shading (Vintage only)
  • SHADING_VERTICAL - Vertical shading (Vintage only)
Note: Not all hybrids are available for all filters. Query camera for filter-specific options.
